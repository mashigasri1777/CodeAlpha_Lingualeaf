# LinguaLeaf Translator

An attractive Language Translation Tool built for the CodeAlpha AI Internship.

## Features
- Translate text between multiple languages
- Auto-detect source language
- Swap languages
- Copy translated text
- Speak translated result
- Responsive UI with dark green, moss green, beige, rosy brown, and midnight green

## Tech Stack
- Flask
- HTML
- CSS
- JavaScript
- Microsoft Translator API

## Project Structure

```bash
├── app.py

│   └── index.html

    ├── css/

    └── js/

```

## Setup
1. Add Translator API keys as environment variables.
2. Install dependencies from `requirements.txt`.
3. Run `python app.py`.

## Output
Open `http://127.0.0.1:5000`
